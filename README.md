A simple website for a local HVAC and plumbing company.

## Installation

Download the project

```bash
gitclone https://github.com/teamallnighter/Excel-Air-Mechanical.git
```

### Install the project

```bash
npm install && npm run init && npm run dev
```

### Other Scripts

#### Run netlify build

```bash
npm run build
```

#### Deploy

```bash
npm run deploy
```

#### Build and Deploy

```bash
npm run build-deploy
```
